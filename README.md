# ATS Resume Co-Pilot

An AI-powered Applicant Tracking System (ATS) that parses resumes, scores them against Job Descriptions, and provides actionable feedback.

## Architecture
- **Frontend:** Streamlit
- **Backend:** FastAPI
- **Intelligence:** Groq LLM API
- **Database/Auth:** Supabase

## Features
- AI-based resume parsing (PDF/DOCX)
- Automated ATS scoring & JD comparison
- Skill validation against project experience
- Secure user authentication via Supabase
